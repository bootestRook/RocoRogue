# RocoRogue

This workspace now keeps only the original public frontend mirror, renamed and branded for RocoRogue.

## Run

Double-click `run.bat`, or run:

```powershell
cd F:\RocoRogue\rocorogue-public
python -m http.server 4173 --bind 127.0.0.1
```

Then open:

```text
http://127.0.0.1:4173/?v=1#/team
```
